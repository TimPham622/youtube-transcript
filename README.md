# YouTube Transcript Copier

Chrome extension that extracts the English transcript from a YouTube video and copies it to your clipboard as plain text.

## Installation

This extension is not currently published in the Chrome Web Store. Install it manually as an unpacked extension.

1. Download

2. Open Chrome and go to:

   ```text
   chrome://extensions
   ```

3. Turn on **Developer mode** in the top-right corner.

4. Select **Load unpacked**.

5. Choose the folder containing:

   ```text
   manifest.json
   popup.html
   popup.js
   ```

## Requirements

* Google Chrome or another Chromium-based browser with Manifest V3 extension support
* A standard YouTube video page
* A video with an available transcript
* An English transcript or caption track for the best results
